# ♔ Android Chess Game

A simple offline chess game built with Java in Android Studio.

## ✨ Features
- Two-player and AI mode
- Check & Checkmate detection
- Sound + animation effects
- Modern dark-themed UI

## 🧩 Tech Stack
- Java
- Android SDK
- SQLite (for game history)
- Material Design + CardView
